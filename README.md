# Control de Neveras 🧊

App web para registrar gaseosas por nevera (Coca-Cola, Postobón, Farmanorte), controlar vencimientos y generar reportes. Hecha para uso en Farmanorte.

## Características

- Registro de productos por nevera con cantidad, lote y fecha de vencimiento.
- Semáforo de vencimientos: verde (ok), amarillo (vence en ≤15 días), rojo (vencida).
- Contadores de unidades totales, por vencer y vencidas.
- Filtros por nevera y filtro rápido de productos por vencer/vencidos.
- Reporte imprimible (guardable como PDF) agrupado por nevera.
- Respaldo y restauración de datos en JSON.
- Funciona 100% en el navegador, sin servidor. Los datos se guardan en el dispositivo (localStorage).

## Uso

Abre `index.html` en cualquier navegador, o visita la versión publicada en GitHub Pages.

## Tecnología

HTML, CSS y JavaScript en un solo archivo. Sin dependencias.
